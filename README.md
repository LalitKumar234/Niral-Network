# React + TypeScript + Vite

## Niral Network Task

Follow the steps to setup this project locally

```js
git clone https://github.com/LalitKumar234/Niral-Network/
```

```js
cd Niral-Network
```

```js
npm install
```

```js
npm run dev
```
